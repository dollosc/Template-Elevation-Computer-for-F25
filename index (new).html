計算公式說明：
1. 視準軸高程 = B.M. + 水準尺讀數（初始）
2. 樓層高程 = 由資料庫表取得（例如 L20 = 225.15 m）
3. 附加厚度 = 墊層 + PC層 + 降版（cm）
4. 總厚度 = TYPE厚度 + 附加厚度（cm）
5. 設計高程 = 樓層高程 − 總厚度(m)
6. 水準尺讀數 = 視準軸高程 − 設計高程
